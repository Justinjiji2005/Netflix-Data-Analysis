# Netflix Data Analysis

## Overview
Exploratory data analysis on Netflix's content library using Python.
Analysed 8,800+ titles to uncover trends in content type, country, genre and ratings.

## Key Findings
- Netflix library is 70% Movies and 30% TV Shows
- United States produces the most content on the platform
- Content additions peaked around 2019–2020
- International Movies and Dramas are the most popular genres
- TV-MA is the most common content rating

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook / Google Colab

## Dataset
Source: Kaggle — Netflix Movies and TV Shows dataset (8,800+ titles)
